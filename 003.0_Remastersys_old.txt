Install Remastersys 3.0.4-1:    lookup 3.0.4-2
	In your terminal,

	Code:	sudo su
	Code:	wget -O - http://www.remastersys.com/ubuntu/remastersys.gpg.key | apt-key add -


	Add the following line to the end of the file:

	Code:	nano /etc/apt/sources.list

	Down arrow to bottom of list.

	Code:	#Remastersys Precise
			deb http://www.remastersys.com/ubuntu precise main

	^x and enter     (control key - x, y- key, then enter key)

	Update the repositories and install Remastersys:

	Code:	apt-get update
	Code:	apt-get install remastersys
	Code:	exit

	Just use Update Manager to have the new Remastersys signed repository ready to use!

	For a GUI in lucid and up to and including precise:
	In Synaptic install :
			remastersys-gtk - pygtk gui that should work on lucid and up to and including precise  use this one!
			remastersys-gui - new gui starting with precise - may not work on older versions prior to 11.10

	restart and

	Code:	sudo apt-get update && sudo apt-get upgrade

	Preferences -> Main Menu -> uncheck (hide) Remastersys

The Manual Method:

	As root - issue 'sudo su' in the terminal window prior to the following command.
	Download and apply the repository gpg key.

	Code:	wget -O - http://www.remastersys.com/ubuntu/remastersys.gpg.key | apt-key add -

	Add the following line that corresponds to your version of Ubuntu to your /etc/apt/sources.list

	Code:	#Remastersys Precise
			deb http://www.remastersys.com/ubuntu precise main

Now just apt-get update or reload in Synaptic to have the new Remastersys signed repository ready to use!
